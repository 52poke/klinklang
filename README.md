这是神奇宝贝百科的工具台，可以进行名词转换、层叠样式表转换等便于编辑神奇宝贝百科的操作。
[神奇宝贝百科](http://wiki.52poke.com/)是一部协作共建的，关于神奇宝贝（Pokémon）的在线百科全书。

## 使用方法

1. 安装 Node.js，NPM，Bower 和 SASS。
2. 分别使用 `npm install` 和 `bower install` 安装依赖的组件。
3. `node app` 启动工具台。
